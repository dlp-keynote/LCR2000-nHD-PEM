                         General Notes
-----------------------------------------------------------------------
DLP LightCrafter GUI is QT based. 

Source Code has been tested with following versions of QT,
* QT4.8.4 for Windows (MinGW and VS2010)
* QT5.0.1 for Windows (MinGW and VS2010)
* QT5.1.0 for Windows (VS2012) 

Link to Sources:

The QT4.8.4 w/VS2010 can be obtained from 
1. TI LightCrafter page - http://www.ti.com/tool/dlplightcrafter (or)
2. QT Web page          - http://download.qt-project.org/official_releases/qt/4.8/4.8.4/
                                          
Visit QT Project downloads page for other QT versions - http://qt-project.org/downloads

------------------------------------------------------------------------
                         Building the project using Qt
------------------------------------------------------------------------
* Open Qt Creator

* Create a new GUI project named "LightCrafterGUI" (or Open existing project and browse 
  to the source code location. Copy all the source files)

* Complete the configuration steps. 

* Make sure the compiler is set correctly in Tools--> Options--> Build & Run--> Kits. 

  Configuration instructions can be found here - 
  http://qt-project.org/doc/qtcreator-2.6/creator-project-opening.html

* Copy the content of this folder and over write the newly created project (skip this 
  step if you are opening the files as an existing project)

* Build the project in Qt Creator
